## Example Queries

Copy any of these into the playground above.

**List users with their posts:**
```graphql
query {
  users {
    id
    name
    email
    role
    posts {
      title
      published
    }
  }
}
```

**Filter posts by author:**
```graphql
query {
  posts(where: { author: { email: { _eq: "alice@example.com" } } }) {
    id
    title
    content
    tags {
      name
    }
  }
}
```

**Paginated posts with total count:**
```graphql
query {
  posts(limit: 3, offset: 0) {
    id
    title
    author {
      name
    }
  }
}
```

**Tags with post counts:**
```graphql
query {
  tags {
    id
    name
    slug
  }
}
```

## About This Demo

The demo API is **read-only** — mutations are disabled to keep the data consistent for everyone. The schema and seed data match the [SaaS Blog example](/examples/saas-blog) in the docs, so examples from the guides work directly here.

## Run It Yourself

To run this exact API locally:

```bash
git clone https://github.com/fraiseql/fraiseql-starter-blog
cd fraiseql-starter-blog
cp .env.example .env
docker compose up
```

Open [http://localhost:8080/graphql](http://localhost:8080/graphql) — you get the same API with a fully mutable database you control.
