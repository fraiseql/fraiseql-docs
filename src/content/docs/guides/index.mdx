Comprehensive guides for building, deploying, and optimizing FraiseQL applications.

**New to FraiseQL?** Start with the [Getting Started](/getting-started/introduction) guide for your first steps. Once you have a working API, explore [Custom Queries](/guides/custom-queries) to learn filtering, search, and pagination.

## Getting Started

- **[Custom Queries](/guides/custom-queries)** — Add filtering, full-text search, and cursor-based pagination to queries
- **[Error Handling](/guides/error-handling)** — Return structured errors from SQL functions and handle them in resolvers
- **[Authentication](/guides/authentication)** — JWT, OAuth, API keys, and custom auth middleware

## Architecture & Design

- **[Schema Design](/guides/schema-design)** — Best practices for naming, types, and view organisation
- **[Multi-Tenancy](/guides/multi-tenancy)** — Row-level security and per-tenant isolation patterns
- **[Advanced Patterns](/guides/advanced-patterns)** — Soft deletes, audit trails, and schema versioning
- **[Projection Tables](/guides/projection-tables)** — Materialised read models and CQRS with PostgreSQL
- **[Threaded Comments](/guides/threaded-comments)** — Recursive CTEs and nested GraphQL types

## Operations

- **[Testing](/guides/testing)** — Unit, integration, and contract tests for FraiseQL resolvers
- **[Performance](/guides/performance)** — Query optimisation, indexes, and connection pooling
- **[Performance Benchmarks](/guides/performance-benchmarks)** — Reproducible comparisons vs Hasura, PostGraphile, async-graphql, and strawberry
- **[Deployment](/guides/deployment)** — Docker, Kubernetes, and cloud deployment checklist

## Integration

- **[Observers](/guides/observers)** — Trigger side effects after mutations — send emails, update projections, publish events
- **[Observer & Webhook Patterns](/guides/observer-webhook-patterns)** — Webhooks and fan-out from observer events
- **[Custom Resolvers](/guides/custom-resolvers)** — Override generated resolvers with custom Python logic
- **[Custom Scalars](/guides/custom-scalars)** — Define and serialise custom GraphQL scalar types
- **[Apollo Sandbox Security](/guides/apollo-sandbox-security)** — Lock down GraphQL introspection in production
- **[Federation & NATS Integration](/guides/federation-nats-integration)** — Multi-database federation over NATS
- **[Federation Configuration](/guides/federation-configuration)** — Detailed federation setup and routing rules
- **[Advanced Federation](/guides/advanced-federation)** — Cross-service queries and federated subscriptions
- **[Advanced NATS](/guides/advanced-nats)** — Durable consumers, JetStream, and NATS clustering

## Reference

- **[FAQ](/guides/faq)** — Frequently asked questions with runnable answers
- **[Troubleshooting](/guides/troubleshooting)** — Diagnose and fix common runtime errors
